# CODSOFT Data Science Internship Projects

This repository contains Machine Learning and Data Science projects completed as part of the CODSOFT Data Science Internship. The projects focus on data preprocessing, visualization, machine learning model development, and performance evaluation using real-world datasets.

## Projects

### 1. Titanic Survival Prediction
- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier
- Final Accuracy: 82.12%

### 2. Movie Rating Prediction
- XGBoost Regressor
- Feature Engineering
- Final R² Score: 0.35

### 3. Iris Flower Classification
- Machine Learning Classification
- Exploratory Data Analysis
- Data Visualization

### 4. Sales Prediction
- Regression Analysis
- Sales Forecasting Model
