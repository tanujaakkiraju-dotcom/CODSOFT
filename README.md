# CODSOFT Data Science Internship Projects

This repository contains the Machine Learning and Data Science projects completed during my CODSOFT Data Science Internship. These projects involve data preprocessing, visualization, feature engineering, model building, and performance evaluation using real-world datasets.

---

## Projects Included

### 1. Iris Flower Classification

A Machine Learning classification project that predicts iris flower species based on sepal and petal measurements.

**Technologies Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

**Key Highlights**
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Classification Model Development
- Model Evaluation
